decode-with-header!

(decode-with-header! codec input)(decode-with-header! codec input header)

Reads a multicodec header from the input stream and checks it against the one provided. Throws an exception if the header does not match, otherwise uses the codec to read a value from the stream.

If a header is not given, the codec’s header is used.

Decoder

protocol

A decoder reads binary sequences and interpretes them as Clojure values.

members

decodable?

(decodable? codec header)

Returns true if the codec supports decoding of the given header.

decode!

(decode! codec input)

Reads bytes from the input stream and returns the read value. May not fully consume the stream if multiple items are present. Throws an exception if the input is malformed or incomplete.

encode-with-header!

(encode-with-header! codec output value)(encode-with-header! codec output header value)

Writes a header to the output stream, then writes out the encoded value. Returns the number of bytes written.

If a header is not given, the codec’s header is used.

Encoder

protocol

An encoder converts values to binary sequences and writes the results to an output stream.

members

encodable?

(encodable? codec value)

Returns true if the value type can be encoded by this codec.

encode!

(encode! codec output value)

Write the value as a sequence of bytes to the output stream. Returns the number of bytes written. Throws an exception if the value cannot be encoded.
